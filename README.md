 # SEC+ Quiz CLI Tool                                                                                                                                                      
   
  A command-line quiz app for studying the CompTIA Security+ exam. Parses practice test files and quizzes you with randomized questions.                                    
                  
  ## How to Run                                                                                                                                                             
                  
  **Requirements:** Python 3

  ```bash
  git clone https://github.com/thothredo/-SEC-Quiz-CLI-Tool-for-Ports.git
  cd -SEC-Quiz-CLI-Tool-for-Ports
  python3 -m venv venv                                                                                                                                                      
  source venv/bin/activate
  pip install flask                                                                                                                                                         
  python3 app.py  

  Features

  - Randomized questions from SEC+ practice tests
  - Score tracking
  - Domain filtering
